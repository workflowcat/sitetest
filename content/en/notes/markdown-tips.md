Markdown is the lingua franca of digital note-taking. Here are tips for effective usage.

## Basics

```markdown
# Heading 1
## Heading 2
### Heading 3

**bold** and *italic*

- Bullet list
- Another item

1. Numbered list
2. Second item

[Link text](https://example.com)
[[wikilink|Internal link]]
```

## Pro Tips

### Use Descriptive Links

> [!tip] Tip
> Instead of "click here", write descriptive links like "learn more about [[en/notes/pkm-intro|knowledge management]]".

### Embrace Simplicity

Markdown's power is in its constraints:

- One idea per paragraph
- Use headings for structure
- Let whitespace breathe

### Code Blocks with Syntax Highlighting

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

```python
def greet(name):
    return f"Hello, {name}!"
```
